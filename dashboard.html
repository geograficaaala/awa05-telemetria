Nodo AWA05

Bienvenido Jose Toriello

Este tablero concentra el nivel de agua, todas las variables expuestas por la estación meteorológica WS-2000 y los indicadores comparativos necesarios para revisión rápida del estado del sitio. Los enlaces fueron corregidos para publicación en GitHub Pages usando index.html y dashboard.html en la raíz del repositorio.
