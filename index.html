Sistema AWA05 de telemetría ambiental y monitoreo hídrico

Sistema de monitoreo continuo de producción de la cosechadora de aire AWA05. Correlaciona nivel de agua con humedad, temperatura, punto de rocío y viento para predecir y optimizar la producción hídrica bajo una interfaz visual de estilo limpio, ligero y orientado a variables ambientales.

Arquitectura del sistema

Funcionamiento del sistema

Pipeline automatizado desde el sensor hasta la nube, ejecutándose 24/7 sobre Raspberry Pi 4B con Debian Trixie. La estructura visual mantiene una jerarquía tipográfica clara y una paleta pastel azul, celeste y blanco coherente con la temática del proyecto.

01 -
Sensor JSN-SR04T
Ultrasonido para medición de distancia al espejo de agua mediante los GPIO 17 y 18. Convierte la lectura a litros con geometría del recipiente: Ø27 cm, altura 50 cm y altura máxima útil de 42.5 cm.
02 -
Estación WS-2000
Transmisión de variables climáticas por HTTP POST al servidor Flask en el puerto 7777 para captura, almacenamiento y posterior correlación con la producción de agua.
03 -
Almacenamiento CSV
Persistencia de datos en nivel_raw.csv y clima_raw.csv dentro de data/raw/. El scheduler ejecuta la rutina cada 60 minutos bajo supervisión de systemd.
04 -
GitHub y Pages
Subida automática programada a las 12:00 y 00:00. El dashboard estático consume el histórico publicado y actualiza visualizaciones y métricas operativas.
Variables monitoreadas

Variables del sistema

Nivel de Agua
JSN-SR04T · GPIO 17/18
Distancia ultrasonido transformada a volumen en litros a partir de la sección circular del recipiente y la altura efectiva de almacenamiento.
Humedad Relativa
WS-2000 · HTTP POST
Variable principal de correlación debido a la dependencia directa entre humedad ambiental y potencial de condensación del sistema.
Temperatura
WS-2000 · HTTP POST
Temperatura del aire en °C. Junto con la humedad relativa, determina el gradiente termodinámico de interés para la producción hídrica.
Punto de Rocío
WS-2000 · Calculado
Indicador derivado crítico para estimar el umbral de condensación y evaluar el rendimiento potencial de la cosechadora de aire.
Velocidad del Viento
WS-2000 · HTTP POST
Velocidad y dirección del flujo de aire. Influye en el intercambio de masa de vapor hacia las superficies de condensación.
Correlación Matemática
Cálculo local · Regresión
Aplicación de modelos de regresión lineal y polinomial entre nivel de agua y variables climáticas para análisis exploratorio del comportamiento del sistema.
